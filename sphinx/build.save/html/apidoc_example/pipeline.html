pipeline.genome_parsing(input_path, **kwargs)¶

The entire pipeline of SeMPI 2.0, callable as a subprocess, including a timeout handler, default kwargs and logging !!

  • sets the default kwargs

  • activates the logging protocol

This func can parse multiple records in fasta and gbk, for each record the kwargs are adjusted and the single_record_genome_parsing is started
