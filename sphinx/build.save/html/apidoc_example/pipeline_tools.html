pipeline_tools.module_prediction.pks_reduction_shift(df)¶

Note: Reduction shift is applied to nrps and pks, as the pks can propagate the reduction to the nrps

The reduction pattern of one module is always applied to the substrate of the previous module, therefore the reduction defined in the modules needs to be shifted by one module

load hydroxly
|

KS-AT-ACP-KS-AT-KR-ACP

shift

hydroxly undefined
|

KS-AT-ACP-KS-AT-KR-ACP
