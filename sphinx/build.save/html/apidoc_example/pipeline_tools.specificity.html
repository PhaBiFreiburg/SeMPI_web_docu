add_seqs(seq_input, input_type='df', min_records=True, **kwargs)¶

Convert new seqs to a form which can be predicted

Parameters:
seq_inputvariable (str, SeqRec Bio, df)

The input to be predicted, parsed based on the input_type arg

input_typestr

Input type for the prediction, can be a df of the pure seqs, a record of the seqs or a path to the seqs (fasta)

min_recordsbool

If min records is used only a small subset (each class is represented ones) of the original alignment is used for the realignment, this is much faster then using for example an entire alignment of 2000 seqs, but the resulting realigned alignment is slightly different, initial tests showed, that the influence on the prediction is minimal.

assign_undefined()¶

assignes classes which can not be predictied (using default values of the function) as undefined classes
