pipeline_tools.rdkit_utils.DB_scoring.substructure_match(target_mol, query_mols, bondCompare=False, verbose=False, ss_limit=20, **kwargs)¶

Matches multiple query_mols to the target_mol as substructs, the score represents the number of indices covered by the query_mols

For each query all possible matches are computed, then reduced targets are created and the next query is matched for this target recursively, this leads to all possible matches of querys in the mol

target_molmol

Mol to match the querys

query_molslist of mols

Query mols

bondComparebool

Include a bond comparison to the MCS algo, (can lead to much smaller scaffolds)

verbosebool

Prints the process of the recursive function

ss_limitint

Number of possible matches per target, some mols can have a lot, therefore should be limited

Scoreslist of tuples

The tuples contain (score, (indices of matches)); sorted by score

Returns a sorted list, of score, combination pairs
